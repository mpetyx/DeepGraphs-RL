Module Hierarchy

  • RDFClosure: This module is brute force implementation of the 'finite' version of RDFS semantics and of OWL 2 RL on the top of RDFLib (with some caveats, see below).
    • RDFClosure.AxiomaticTriples: Axiomatic triples to be (possibly) added to the final graph.
    • RDFClosure.Closure: The generic superclasses for various rule based semantics and the possible extensions.
    • RDFClosure.CombinedClosure: The combined closure: performing both the OWL 2 RL and RDFS closures.
    • RDFClosure.DatatypeHandling: Most of the XSD datatypes are handled directly by RDFLib.
    • RDFClosure.Literals: Separate module to handle literals.
    • RDFClosure.OWL': OWL and OWL2 terms.
    • RDFClosure.OWLRL: This module is a brute force implementation of the OWL 2 RL profile.
    • RDFClosure.OWLRLExtras: Extension to OWL 2 RL, ie, some additional rules added to the system from OWL 2 Full.
    • RDFClosure.RDFS': RDF(S) terms.
    • RDFClosure.RDFSClosure: This module is brute force implementation of the RDFS semantics on the top of RDFLib (with some caveats, see in the introductory text).
    • RDFClosure.RestrictedDatatype: Module to datatype restrictions, ie, data ranges.
    • RDFClosure.XsdDatatypes: Lists of XSD datatypes and their mutual relationships
